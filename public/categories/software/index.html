My Stack of Tools for 2017

Feb 10, 2017 in software

I adopted Node JS for my projects and on the way, I also discovered Angular and Sails JS and finally for Databases: DynamoDB or MongoDB work perfectly for most projects. I am using a lot DynamoDB for convinience, as I have also adopted Amazon Web Services as the cloud platform for most of my projects.

For blogging engines Hexo is pretty cool, which is what I am using on this blog, it does not have all the feature of wordpress, but it does the job nicely and is a good alternative.

On the other hand, I’m also using a lot: Apache 2, PHP7, MariaDB and CakePHP as a framework.
